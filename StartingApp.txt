Start a project::::
1) Make a project folder
2) Run "npm init"
3) Run "npm install express ejs mongoose body-parser --save"
4) Make app.js
5) Inside app.js make basic express importing "var express = require("express");"
6) Then the app variable "var app = express();"
7) Add "var bodyParser = require("body-parser");"
8) Add "app.set("view engine", "ejs");" so that we don't have to write ".ejs" in "res.render()" calls.
9) And to tell express to use it add "app.use(bodyParser.urlencoded({extended: true}));"
10) Add "app.listen(3000, function(){...});" code
11) Add Homepage i.e. "/" by this code "app.get("/", function(req, res){});" to check it's working just include "res.send("HomePage");"
12) Make a view directory and start adding .ejs files
13) To check routes are calling ejs perfectly run "res.render("homepage");" in step 9.
14) Now run the server "node app.js" and check if everything is working fine. 

Note: ejs 3.1.3 include partial header/footer is changed to "<%- include('partials/header') %>"
Note: Use this mongoose syntax: "mongoose.connect("mongodb://localhost:27017/yelp_camp", {useUnifiedTopology: true, useNewUrlParser: true});"
